# Web 管理界面使用指南

本项目提供了一个简洁美观的 Web 管理界面，让您可以通过浏览器轻松管理所有提醒任务。

## 访问界面

部署成功后，直接访问你的 Worker URL：

```
https://your-worker-name.your-account.workers.dev
```

或者如果配置了自定义域名：

```
https://reminders.yourdomain.com
```

## 功能介绍

### 1. 登录验证

首次访问会看到登录页面：

- **管理员密钥输入框**：输入您在部署时设置的 `ADMIN_TOKEN`
- **登录按钮**：验证密钥并进入管理界面

> 💡 提示：密钥会保存在浏览器本地存储，下次访问无需重新登录

### 2. 添加新提醒

登录后，界面顶部显示"添加新提醒"表单：

**必填字段：**
- **提醒标题**：简短描述提醒内容（例如：登录 DigitalPlat）
- **间隔天数**：每隔多少天发送一次（例如：180）
- **提醒内容**：详细的提醒信息（例如：请登录 DigitalPlat 网站，避免账号过期）
- **接收邮箱**：接收提醒的邮箱地址

**可选字段：**
- **相关链接**：提醒相关的网址（将显示在邮件中）

填写完成后点击"创建提醒"按钮即可。

### 3. 提醒列表

界面中部显示所有已创建的提醒任务。每个提醒卡片包含：

**信息展示：**
- 提醒标题（大字体显示）
- 启用状态（绿色=已启用，红色=已禁用）
- 提醒内容描述
- 接收邮箱
- 间隔天数
- 已发送次数
- 下次发送时间（智能显示：今天/明天/X天后）
- 相关链接（如有）

**操作按钮：**
- **✏️ 编辑**：修改提醒的任何信息
- **⏸️ 禁用 / ▶️ 启用**：快速切换提醒状态
- **🗑️ 删除**：永久删除提醒（需确认）

### 4. 编辑提醒

点击"编辑"按钮会弹出编辑对话框：

- 所有字段都会预填充当前值
- 修改所需字段
- 点击"保存修改"确认，或"取消"放弃

### 5. 手动触发检查

界面底部有"手动触发检查"功能：

- 点击"立即检查"按钮
- 系统会立即检查所有到期的提醒
- 显示检查结果统计：
  - 检查数量
  - 发送成功数量
  - 发送失败数量
  - 错误详情（如有）

> 💡 用途：测试邮件发送、立即发送到期提醒、验证配置是否正确

### 6. 退出登录

点击界面底部的"退出登录"按钮：

- 清除本地存储的密钥
- 返回登录页面

## 界面特点

### 🎨 设计特色

- **渐变背景**：紫色系渐变，视觉舒适
- **卡片式布局**：信息清晰，层次分明
- **响应式设计**：支持手机、平板、桌面端
- **交互反馈**：按钮悬停效果、动画过渡

### 📱 移动端适配

- 自动适应小屏幕
- 按钮和表单优化触摸操作
- 文字大小和间距适合移动设备

## 常见问题

### Q: 忘记了 ADMIN_TOKEN 怎么办？

A: 需要通过 Cloudflare Dashboard 或 Wrangler CLI 重新设置：

```bash
npx wrangler secret put ADMIN_TOKEN
```

### Q: 可以修改界面样式吗？

A: 可以！修改 `public/styles.css` 文件，然后重新部署：

```bash
npm run deploy
```

### Q: 界面支持哪些浏览器？

A: 支持所有现代浏览器：
- Chrome / Edge (推荐)
- Firefox
- Safari
- 移动端浏览器

### Q: 可以设置多个管理员吗？

A: 当前版本只支持一个 ADMIN_TOKEN。所有知道 token 的人都可以管理提醒。

### Q: 界面语言可以切换吗？

A: 当前版本仅支持中文。如需英文版，可以修改 `public/index.html` 和 `public/app.js` 中的文本。

## 安全建议

1. **使用强密码作为 ADMIN_TOKEN**
   - 至少 20 个字符
   - 包含字母、数字、符号

2. **不要在公共场所登录**
   - Token 会保存在浏览器中
   - 使用公共电脑后记得退出登录

3. **定期更换 Token**
   - 定期更新 ADMIN_TOKEN 提高安全性

4. **考虑使用自定义域名**
   - 更容易记忆
   - 可以配置 HTTPS（Cloudflare 自动提供）

## 快捷键

浏览器快捷键：
- `Esc` - 关闭编辑对话框
- `F5` - 刷新提醒列表
